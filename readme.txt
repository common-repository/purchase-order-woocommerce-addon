== Description ==

This plugin adds a purchase order in WooCommerce for customers to complete the order.



= Features =
1. Very Simple Clean Code plugin to add a Purchase Order payment method to woocommerce
2. No technical skills needed.
3. Prerequisite visualized on screenshots.
4. Adds PO Number and PO Note to Order details

== Installation ==

1. Upload 'woo-purchase-order-addon' folder to the '/wp-content/plugins/' directory
2. Activate 'Purchase Order WooCommerce Addon' from wp plugin lists in admin area
3. Plugin will appear in settings of woocommerce
4. You can set the addon settings from wocommmerce->settings->Checkout->Purchase Order Settings
